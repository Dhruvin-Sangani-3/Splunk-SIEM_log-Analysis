# Splunk-logs-analysis

## Adding Data into Splunk Enterprise

This project demonstrates how to upload and verify log data in Splunk Enterprise as part of a beginner Splunk Proof of Concept (PoC).

---

# Objective

The objective of this exercise is to:

- Install Splunk Enterprise
- Upload log data into Splunk
- Verify successful log ingestion
- Practice basic SPL queries
- Document the complete process

---

# Step 1 – Login to Splunk

Download and install Splunk Enterprise from the official website.

After installation:

1. Open Splunk Web
2. Login using your credentials
